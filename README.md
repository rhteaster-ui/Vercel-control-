# Vercel Control

Blueprint-based Next.js App Router project for Vercel deployment manager + browser IDE.

## Required Environment Variables

- `UPSTASH_REDIS_REST_URL`
- `UPSTASH_REDIS_REST_TOKEN`
- `ADMIN_SECRET_KEY`
- `VERCEL_TOKEN`
- `VERCEL_TEAM_ID`

## Development

```bash
npm install
npm run dev
```
